### Semver kararı
Export yüzeyi değişti (kova ekleme/taşıma). Tüketici import yolları kırılıyorsa **major** (`0.5.0 → 1.0.0` veya `0.5.0 → 0.6.0` pre-1.0 konvansiyonuyla minor=breaking). Paket hâlâ `0.x` (pre-1.0): SemVer'de 0.x'te minor bump breaking sayılır → **`0.5.0 → 0.6.0` minor yeterli ve doğru**. Karar gerekçesi commit mesajına yazılır. (Eğer ekip 1.0 stabil API ilan etmek istiyorsa major=`1.0.0`; bu Açık Soru olarak işaretli.)
