# Premium Web Design

Detayli tasarim kurallari .claude/rules/servicecore-tasarim.md'de. Bu skill ek prensipler:

## Layout Ritmi

- Strict grid kullan ama bilincli kir (ilgi yaratmak icin)
- Agresif whitespace — premium = nefes alacak alan
- Bento grid: ozellik vitrinleri icin
- Section padding comert: en az 96px dikey

## Gorsel

- Yuvarlak koseler: rounded-2xl veya rounded-3xl
- Subtle parallax arka planlarda derinlik katar
- Gorsel kalitesi yuksek, optimize edilmis

## CTA

- Her section'da net bir CTA olmali
- Max 2 buton yan yana

## Troubleshooting

### Flat gorunen tasarim
En az bir glassmorphic element + gradient + glow ekle. Flat UI = Basarisizlik.

### Kalabalik layout
Whitespace artir. Section padding 96px+, kart padding 32px+.
